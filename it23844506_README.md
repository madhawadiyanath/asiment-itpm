# Test Automation Project

## Overview
This project is an automated test framework built using Playwright for testing the Pixels Suite Chat Translator application. It provides automated testing capabilities for Singlish to Sinhala translation functionality.

## Project Details

**Repository:** https://github.com/madhawadiyanath/asiment-itpm.git

**Author:** Swarnakantha L M D

**Assignment:** ITPM Assignment

## Purpose
The test automation framework is designed to:
- Automate test case execution for chat translator functionality
- Read test cases from Excel spreadsheets
- Execute tests against the Pixels Suite Chat Translator frontend
- Validate actual outputs against expected results
- Generate automated test reports

## Key Features
- **Playwright Integration:** Uses Playwright's sync API for browser automation
- **Excel-based Test Cases:** Supports reading test cases from Excel files
- **Flexible Column Detection:** Auto-detects input, expected output, and actual output columns
- **Status Tracking:** Tracks and reports test results
- **Configurable:** Command-line arguments for custom configurations

## Technologies Used
- **Python 3.x**
- **Playwright** - Browser automation
- **OpenPyXL** - Excel file handling
- **Sync API** - Synchronous test execution

## Configuration
Default configurations include:
- **Frontend URL:** `https://www.pixelssuite.com/chat-translator`
- **Excel File:** `Assignment 1 - Test cases.xlsx`
- **Sheet Name:** ` Test cases`
- **Supported Input Columns:** Singlish, Input, Singlish Input, Test Input, Source, Sentence, Text
- **Supported Expected Output Columns:** Sinhala, Expected_Output, Expected Output, Expected output, Expected, Expected Sinhala
- **Supported Actual Output Columns:** Actual_Output, Actual Output, Actual output, Actual

## Installation & Setup
1. Install Python dependencies:
   ```bash
   pip install playwright openpyxl
   ```

2. Install Playwright browsers:
   ```bash
   playwright install
   ```

3. Prepare test case Excel files in the project directory

## Usage
Run the test automation framework:
```bash
python test_automation.py
```

With custom parameters:
```bash
python test_automation.py --excel-file <path> --sheet-name <sheet> --frontend-url <url>
```
